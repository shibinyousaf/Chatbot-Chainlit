# Chatbot-Chainlit

## How to Run?

1.Create a Virtual Environment

```bash
conda create -n llmapp python=3.10 -y

```

1.Activate Virtual Environment

```bash
conda activate llmapp 

```

1.Install the requirement Packages

```bash
pip install -r requirements.txt

```

## Chainlit Commands

```bash
chainlit init

```

```bash
chainlit run app.py
```
